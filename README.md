# BerlengasPass 🌊

Plataforma de reservas para o Arquipélago das Berlengas.

## Desenvolvimento local

```bash
npm install
npm run dev
```

## Deploy no Vercel

### Opção 1 — Git push automático
1. Cria um repositório no GitHub (ou GitLab)
2. Faz push desta pasta para o repo
3. Liga o repo ao Vercel em [vercel.com/new](https://vercel.com/new)
4. O Vercel deteta automaticamente o Vite — deixa as definições por omissão

### Opção 2 — Vercel CLI
```bash
npm install -g vercel
vercel
```

## Build manual

```bash
npm run build
# Output em /dist — pronto para servir
```
